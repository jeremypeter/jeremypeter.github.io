Automation

with Gulpjs

What does Gulpjs do for us?

  1. Watches for file changes
  2. Checks for HTML errors
  3. Compiles partials and metadata
  4. Auto-append tracking codes to urls
  5. Inlines css
  6. Minifies code
  7. Encodes special characters
  8. Auto-refreshes browser
  9. Tests code
  10. Creates final output file (email.html)
  11. Optional: send tests to Litmus
  12. Optional: send to email address
